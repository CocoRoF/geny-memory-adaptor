# Eval fixture — MIRACL-ko (dev) derivative

Derived from [MIRACL](https://huggingface.co/datasets/miracl/miracl) /
[miracl-corpus](https://huggingface.co/datasets/miracl/miracl-corpus)
(Apache-2.0). 213 dev queries, 2,835 human-judged Korean Wikipedia passages
(all positives + judged hard negatives), 3,057 graded qrels. Rebuilt with
`eval/build_corpus.py`; perturbation variants generated by `eval/perturb.py`.
Redistributed under the same Apache-2.0 license with attribution to the
MIRACL authors.
